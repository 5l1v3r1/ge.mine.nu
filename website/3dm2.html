Ebuild for 3ware "3dm2-10.2" (tested with "3ware Inc 9550SX SATA-RAID")

3dm2-10.2.ebuild

Installation:
mkdir -p /root/OVERLAY/sys-block/3dm2/files
wget "http://bugs.gentoo.org/attachment.cgi?id=241017" -O /root/OVERLAY/sys-block/3dm2/files/3dm2.init
wget "http://bugs.gentoo.org/attachment.cgi?id=241013" -O /root/OVERLAY/sys-block/3dm2/files/LSI.txt
wget "http://bugs.gentoo.org/attachment.cgi?id=269835" -O /root/OVERLAY/sys-block/3dm2/3dm2-10.2.ebuild
cd /root/OVERLAY/sys-block/3dm2
PORTDIR_OVERLAY="/root/OVERLAY"
ebuild 3dm2-10.2.ebuild digest merge

/etc/init.d/3dm2 start
rc-update add 3dm2 default

Uninstall:
/etc/init.d/3dm2 stop
emerge -C 3dm2
rm -fr /etc/3dm2

I' pretty sure this will not be added to portage soon!
Read the bugtracker entry about it.
