Ebuild for 3ware "tw_firmware-9.4.0.1"

tw_firmware-9.4.0.1.ebuild

Installation:
mkdir -p /root/OVERLAY/sys-block/tw_firmware
wget "http://bugs.gentoo.org/attachment.cgi?id=113935" -O /root/OVERLAY/sys-block/tw_firmware/tw_firmware-9.4.0.1.ebuild
PORTDIR_OVERLAY="/root/OVERLAY"

cd /root/OVERLAY/sys-block/tw_firmware
wget http://www.3ware.com/download/Escalade9650SE-Series/9.4.0.1/9.4.0.1-9650SE-Upgrade.zip -O /usr/portage/distfiles/9.4.0.1-9650SE-Upgrade.zip

ebuild tw_firmware-9.4.0.1.ebuild digest
ebuild tw_firmware-9.4.0.1.ebuild merge

Uninstall:
emerge -C tw_firmware

I hope this will be added to portage soon!
Read the bugtracker entry about it.
