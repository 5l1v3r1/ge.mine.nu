anschlussinfo

Script that shows user & DSL information on a german phone number you specify.
It uses formulars on www.dasoertliche.de and www.olis.de. 
[user@ge.mine.nu ~/]$ ./anschlussinfo 0441 884480

0441-884480
DSL 3000
MobilCom Shop
Elisenstr. 5, 26122 Oldenburg Oldb

[user@ge.mine.nu ~/]$ ./anschlussinfo 0441 44088

0441-44088
kein DSL
TTH-Transporttaxe GmbH
Bahnhofsallee 168, 26133 Oldenburg Oldb

... have fun! ;)
